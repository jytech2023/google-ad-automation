---
name: Google Ads Tag Location
description: Where to find Google Ads tag/conversion IDs in the Google Ads UI (2026 layout)
type: reference
---

Google Ads tag setup is found under **Google Ads > Tools > Data manager > Google tag** (not under Goals > Conversions > Tag setup as older docs suggest).

Click "Configure" on the Google tag to find the Tag ID (format `AW-XXXXXXXXX`).
